Box Models And CSS Definitions

CSS Source Order

sets the order to lay out an item in a flex or grid container.

CSS Value

used to define the set of valid values for a CSS property or function.

CSS Selector

terms that tell the browser which HTML elements should be selected.

CSS Property

determines an HTML element's style or behavior.

CSS Declaration

key-value pair of a CSS property and its value.

CSS Cascade

the logic your browser uses to determine which CSS rulesets are the most important, especially when they conflict with each other.

CSS inheritance

controls what happens when no value is specified for a property on an element.

Class selector

selects elements with a specific class attribute.

ID Selector

use the id attribute of an HTML element to select a specific element.

CSS Rule

set contains one or more selectors and one or more declarations.

CSS Specificity

an algorithm used by browsers to determine the CSS declaration that is the most relevant to an element.

CSS

CSS is a stylesheet language used to describe the presentation of a document written in HTML or XML.

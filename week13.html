Uniform Resource Locator (URL) the address of a specific location on the web.
Firewall A firewall is something that helps computer from viruses and other stuff.
Transmission Control Protocol-Internet Protocol (TCP-IP) a suite of communication protocols used to interconnect network devices on the internet.
Network Protocols Network protocols are established sets of rules that govern how data is transmitted between devices on a network, enabling communication even with differences in internal processes or design.
Computer Network interconnected computing devices that can exchange data and share resources with each other.
IPv4 Pv4 is like the address system for the internet.Every device that connects to the internet, like your phone or laptop gets a unique IPv4 address kind of like a home address so data knows where to go.
IPv6 Pv6, or Internet Protocol version 6, is the latest version of the Internet Protocol, designed to address the limitations of its predecessor, IPv4, by providing a much larger address space and other improvements.
Domain Name Services (DNS) Turns domain names into IP addresses, which allow browsers to get to website and other internent resources.
Hyper-text transfer protocol (HTTP/HTTPS) A protocol that defines how web browere and servers exchange data.
File Transfer Protocol a method for moving files between computers on a network.
Cloud Computing the practice of using a network of remote servers hosted on the internet to store, manage, and process data, rather than a local server or a personal computer.
OSI Model a conceptual framework that divides network communications functions into seven layers.
Virtual Private Network (VPN) establishes a digital connection between your computer and a remote server owned by a VPN provider.
